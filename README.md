# 🌄 Projeto Parallax

Projeto desenvolvido durante o curso **Desenvolvimento Web Completo** da Udemy, com o objetivo de praticar técnicas de HTML5 e CSS3 por meio da implementação do efeito visual Parallax.

## 📷 Demonstração

**URL** -> <a href="https://maicondante.github.io/Projeto_Parallax/">Projeto Parallax</a>

---

## 📖 Sobre o Projeto

O Projeto Parallax foi desenvolvido para demonstrar a criação de efeitos visuais modernos utilizando exclusivamente HTML5 e CSS3.

A proposta consiste em construir uma página composta por seções de conteúdo intercaladas com imagens de fundo que utilizam o efeito Parallax, criando uma sensação de profundidade e dinamismo durante a rolagem da página.

Todo o projeto foi desenvolvido sem a utilização de JavaScript ou bibliotecas externas, destacando o potencial do CSS para criação de experiências visuais atrativas.

---

## 💻 Tecnologias Utilizadas

* HTML5
* CSS3
* Folha de estilo externa (`style.css`)

---

## 🎨 Recursos Implementados

O projeto contempla recursos como:

* Efeito Parallax utilizando apenas CSS
* Imagens de fundo em tela cheia
* Estruturação semântica do conteúdo
* Seções organizadas para melhor experiência de leitura
* Layout limpo e moderno
* Separação entre estrutura (HTML) e apresentação (CSS)
